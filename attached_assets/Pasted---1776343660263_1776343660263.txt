أريد تعديل لوحة الأدمن بحيث تحتوي على زرين منفصلين لكل مشترك، وليس زرًا واحدًا يجمع الإلغاء والاسترجاع.

نفذ التالي فقط:

1. في صفحة /admin/overview (أو جدول آخر المشتركين):

   * أضف زرين لكل مشترك:

     1. "إلغاء الاشتراك"
     2. "استرجاع المبلغ"

2. الزران يجب أن يكونا ظاهرين فقط للأدمن

   * ولا يظهرا للمستخدمين العاديين

3. زر "إلغاء الاشتراك":

   * عند الضغط عليه افتح نافذة تأكيد بسيطة
   * بعد التأكيد:

     * أرسل طلب إلى endpoint جديد خاص بالأدمن
     * هذا endpoint يقوم فقط بـ:
       UPDATE subscriptions
       SET status = 'cancelled'
       WHERE id = subscription_id
   * لا يرسل أي طلب إلى Moyasar
   * لا يقوم بأي refund
   * بعد النجاح:

     * حدّث الجدول مباشرة في الواجهة
     * أظهر رسالة:
       "تم إلغاء الاشتراك بنجاح"

4. زر "استرجاع المبلغ":

   * عند الضغط عليه افتح نافذة تأكيد مستقلة
   * بعد التأكيد:

     * أرسل طلب إلى endpoint جديد خاص بالأدمن
     * هذا endpoint يقوم فقط بعملية refund عبر Moyasar API
   * إذا نجح refund:

     * حدّث قاعدة البيانات بحقول واضحة مثل:
       refund_status = 'refunded'
       refunded_at = now()
       refund_amount_halalas
     * لا تغيّر status إلى cancelled تلقائيًا إلا إذا كان هذا منطقيًا في نظامك الحالي وتم توضيحه
   * إذا فشل refund:

     * لا تغيّر حالة الاشتراك
     * وأظهر رسالة خطأ واضحة

5. لا تجعل الزرين مرتبطين ببعض:

   * يمكن إلغاء الاشتراك بدون استرجاع
   * ويمكن استرجاع المبلغ بشكل منفصل
   * ولا يتم تنفيذ أحدهما تلقائيًا عند الضغط على الآخر

6. إذا كان لدى الاشتراك moyasar_payment_id غير موجود:

   * زر "استرجاع المبلغ" يجب أن يكون معطلًا أو يظهر رسالة واضحة أن الاسترجاع غير متاح

7. لا أريد الآن:

   * partial refunds
   * workflow معقد
   * logs متقدمة
   * notifications
   * أي تغييرات على UI خارج الجدول والنافذة البسيطة

8. لا تغيّر أي شيء آخر في النظام الحالي

   * لا تغيّر قاعدة البيانات إلا الحقول اللازمة للاسترجاع إذا لم تكن موجودة
   * لا تغيّر نظام الاشتراك أو الدفع الحالي
   * لا تضف ميزات إضافية

9. بعد التنفيذ أعطني فقط:

   * أسماء الـ endpoints الجديدة
   * الملفات التي عدلتها
   * كيف يعمل زر إلغاء الاشتراك
   * كيف يعمل زر استرجاع المبلغ
   * ومتى يتغير status ومتى لا يتغير
