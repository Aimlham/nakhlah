أريد إضافة ميزة في لوحة الأدمن لتفعيل حساب يدويًا على باقة مدفوعة بدون دفع، مع تمييزه بوضوح أنه تفعيل يدوي وليس عملية دفع.

المطلوب:

1. في صفحة /admin/overview أو صفحة إدارة الاشتراكات:

   * أضف زر:
     "تفعيل يدوي"

2. الزر يسمح لي بإدخال:

   * email المستخدم
   * مدة التفعيل بالأيام
   * سبب التفعيل، مثل:

     * معلن
     * تجربة
     * شريك
     * دعم

3. عند التفعيل:

   * إذا كان المستخدم موجودًا في auth.users:

     * أنشئ أو حدّث اشتراك له في جدول subscriptions
   * اجعل:

     * plan = 'pro'
     * status = 'active'
     * activation_source = 'manual'
     * manual_activation_reason = السبب
     * amount_halalas = 0
     * moyasar_payment_id = null
     * moyasar_invoice_id = null

4. إذا المستخدم غير موجود:

   * لا تنشئ مستخدم جديد تلقائيًا
   * أظهر رسالة:
     "المستخدم غير موجود، اطلب منه التسجيل أولًا"

5. أضف حقولًا جديدة في subscriptions إذا لم تكن موجودة:

   * activation_source text DEFAULT 'payment'
   * manual_activation_reason text
   * manually_activated_by text
   * expires_at timestamptz

6. إذا تم تحديد مدة التفعيل:

   * احسب expires_at = now() + عدد الأيام
   * وإذا انتهى expires_at، يجب اعتباره غير نشط حتى لو status = active

7. في لوحة إحصائيات الاشتراكات:

   * أظهر نوع الاشتراك:

     * دفع
     * تفعيل يدوي
   * لا تحسب الاشتراكات ذات activation_source = 'manual' ضمن دخل اليوم أو إجمالي الدخل
   * لكن احسبها ضمن "المستخدمين النشطين" أو أضف رقم منفصل:

     * مشتركين مدفوعين
     * تفعيل يدوي

8. في جدول آخر المشتركين:

   * أضف Badge واضح:

     * "مدفوع"
     * "يدوي"
   * إذا يدوي، اعرض السبب

9. لا تغيّر نظام الدفع الحالي.

10. لا تعتبر التفعيل اليدوي كعملية Moyasar.

11. لا تضف رصيد وهمي.

12. لا تضع أي مبلغ في الإيرادات للتفعيل اليدوي.

13. تأكد أن صلاحية الوصول للميزات تعتبر المستخدم يدويًا active إذا:

* status = active
* و activation_source = manual
* و expires_at لم ينتهِ

14. بعد التنفيذ أعطني:

* الحقول التي أضفتها
* الـ endpoint المستخدم للتفعيل اليدوي
* كيف أفعّل مستخدم بالإيميل
* كيف يتم منع احتساب التفعيل اليدوي ضمن الإيرادات
